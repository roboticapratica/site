Configuração do módulo bluetooth HC-06

O módulo HC-06 permite estabelecer uma comunicação via bluetooth com placas Arduino que não dispõem deste recurso.

Parâmetros de fábrica:

  • SSID: HC-06
  • Senha: 1234
  • Velocidade: 9600 bps
  • Frequência: 2.4Ghz
  • Alcance: 10 metros

Para alterar os parâmetros SSID, senha e velocidade vamos usar uma placa Arduino para enviar comandos AT (linguagem de comandos para modems e módulos bluetooth).

Lista de material:

  • 1 Arduino UNO ou compatível
  • 1 módulo bluetooth HC-06
  • 1 resistor de 1K (marrom preto vermelho)
  • 1 resistor de 2K (vermelho preto vermelho)

Montagem do circuito elétrico:

Ligue o módulo HC-06 no Arduino, conforme diagrama abaixo. Os resistores são usados como divisor de tensão para reduzir a tensão de 5V para 3.3V no pino RX do módulo (importante para não danificar o módulo bluetooth).
